# ⚽ Jersey's Land - Marketplace de Camisetas de Futebol

O **Jersey's Land** é uma plataforma de marketplace focada no nicho esportivo, permitindo que usuários comprem e vendam camisetas de futebol de forma simples e integrada. 

O projeto simula a dinâmica de um e-commerce C2C (Consumer-to-Consumer) e B2C, onde entusiastas e colecionadores de futebol podem anunciar seus produtos, gerenciar postagens e explorar o catálogo disponível.

> **Status do Projeto:** 🚀 Concluído (Desenvolvido de forma colaborativa/em grupo)

---

## 🛠️ Tecnologias Utilizadas

O ecossistema do marketplace foi construído utilizando as seguintes tecnologias:

*   **Framework Principal (Backend/MVC):** Ruby on Rails
*   **Frontend:** HTML5, CSS3
*   **Banco de Dados:** Banco relacional para gestão de usuários, anúncios e produtos (estruturado via Rails Active Record)

---

## 🎯 Principais Funcionalidades

*   🛒 **Fluxo de Marketplace:** Lógica estruturada para cadastro, exibição e gerenciamento de anúncios de camisetas de futebol.
*   👥 **Cadastro de Usuários:** Sistema de autenticação para que compradores e vendedores gerenciem seus perfis e postagens na plataforma.
*   🗂️ **Catálogo de Produtos:** Organização e filtragem visual dos itens disponíveis para compra, facilitando a navegação do usuário.

---

## 👥 Competências Desenvolvidas

*   Desenvolvimento ágil de software trabalhando em equipe integrada.
*   Modelagem de dados e aplicação do padrão MVC (Model-View-Controller) com Ruby on Rails.
*   Foco em regras de negócio para e-commerce (relacionamento entre produtos, vendedores e compradores).
